Privacy Notice

This static website is designed to be simple and privacy-conscious.

Static website

The GitHub Pages website uses static HTML and CSS. It does not include backend code, account registration, contact forms, tracking scripts, cookies, or embedded analytics.

External demo link

The "Launch Demo Map" button opens the hosted demo application at watergapmap-mvp.onrender.com. That external application may be served by separate infrastructure and may generate standard server logs needed to operate the service.

Public data

WaterGapMap is built around public water-data review. Displayed records may describe structures, locations, administrative identifiers, and other public-record attributes. The tool should not be used to infer legal status, compliance status, or ownership.
