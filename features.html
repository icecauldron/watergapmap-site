Features

WaterGapMap packages public data signals into a review workflow built for transparency and follow-up.

Interactive structure map

Browse mapped water structures and inspect available public attributes in a geographic context.

Review-priority categories

Visual priority signals help reviewers focus attention on records where available data suggests additional review may be useful.

Activity-status context

Reported-use recency and related data fields can help distinguish current, uncertain, inactive, and no-data situations.

Source-oriented workflow

Records are intended to point reviewers back to public source systems and supporting documentation.

Scope note: The demo is a screening interface. It does not make legal conclusions, assign fault, certify compliance, or establish water-right ownership.
